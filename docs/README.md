# Documentation

The repository follows the Divio documentation split.

- [Tutorials](tutorials/README.md)
- [How-to Guides](how-to/README.md)
- [Reference](reference/README.md)
- [Explanation](explanation/README.md)

Current architectural and design notes live under `docs/explanation/`.
Generated API reference inputs are staged under `artifacts/api-docs/` during build.
The snippet-driven core capability work, including the promoted MIDI package, is documented through the explanation, how-to, and reference pages linked from those indexes.
